# Advance-IP-Scanner
Advance IP Scanner is tool for detect open ip Python3

# Installation
Windows 

```
git clone https://github.com/GiuCoder/Advance-IP-Scanner.git

cd Advance-IP-Scanner

python scanner.py 
```

Linux

```
apt-get install git python3 python3-pip

git clone https://github.com/GiuCoder/Advance-IP-Scanner.git

cd Advance-IP-Scanner

python3 scanner.py 
```
